Introduction:

What is vanbUI?

vanbUI is a UI library for vanilla. This project is made to give nice UIs (ex: shadcn) for vanilla (vanilla HTML, CSS and JS)

How to use:

Using vanbUI is pretty easy. All you have to do is define an attribute vanb="true" and some other attributes like variant, type, etc.

You can also copy paste the components right from this docs. But first you need to add vanbUI css for your page. So use this cdn below to add it to your webpage.

And vanbUI uses font awesome for icons so you need to include font awesome as well. As for the fonts.... well it's upto you.
